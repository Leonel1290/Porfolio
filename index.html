Nuestro Proyecto: Detector de Gas Programable

Hemos desarrollado un sistema autónomo para la **detección temprana de fugas de gas**, diseñado para brindar tranquilidad y seguridad en cualquier entorno.

Utilizamos un **sensor MQ-x** de alta precisión para identificar la presencia de gases peligrosos. El corazón del sistema es un **microcontrolador (ej. Arduino/ESP32)**, programado para leer continuamente los datos del sensor y procesarlos en tiempo real.

Al detectar una concentración de gas que supera los umbrales de seguridad predefinidos, el dispositivo activa **alertas visuales (LEDs)** y **sonoras (buzzer)**. Además, el código está optimizado para una respuesta rápida y un bajo consumo de energía.

La programación se realiza en **C/C++ (para Arduino/ESP32)**, enfocándonos en la eficiencia y la robustez del sistema. Hemos implementado una lógica de control que permite la calibración y el ajuste de sensibilidades, garantizando la fiabilidad de las mediciones.

Este proyecto es una muestra de cómo la **programación** y la **electrónica** pueden combinarse para crear soluciones innovadoras que protegen vidas.
